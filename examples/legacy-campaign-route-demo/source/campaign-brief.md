# Demo Source Brief: Campaign Strategy Route

This synthetic example shows how `tech-route-maker` can support an advertising or campaign-planning route diagram.

## Campaign Goal

Design a campaign route for a new research-tool launch aimed at graduate students, lab managers, and academic teams.

## Source Evidence

- Audience: graduate students need fast paper-to-slide workflows; lab managers need repeatable team reporting; academic teams need editable visuals.
- Insight: users trust tools more when outputs remain editable and evidence-linked.
- Strategy: position the product as an agent-assisted workflow that turns source material into editable diagrams and presentations.
- Creative route: problem hook, evidence-first promise, before/after demo, use-case proof, and call to try the demo.
- Channel route: GitHub README, short video, social posts, lab-group demo deck, and community templates.
- Measurement: GitHub stars, demo runs, template downloads, issue feedback, and conversion to trial users.
- Optimization: refine messaging based on drop-off, issue themes, and demo-output quality feedback.
