# Demo Source Brief: Multimodal PV Defect Detection

This is a synthetic onboarding example for `tech-route-maker`. It is not a real paper.

## Project Goal

Design a method overview figure for a research project on multimodal photovoltaic module defect detection. The figure should help an academic reader understand the method route from image collection to validated defect reports.

## Source Evidence

- Input data: RGB module images and infrared thermal images.
- Preparation: image quality screening, alignment, annotation, augmentation, and train/validation/test split.
- Core method: a baseline detector, multimodal feature fusion, attention-enhanced feature extraction, and loss/metric-driven optimization.
- Training and inference: supervised training, validation monitoring, inference on unseen module images, and confidence filtering.
- Evaluation: comparison against baselines, ablation of fusion/attention modules, precision/recall/mAP, and qualitative defect maps.
- Output: defect category, location, confidence, and report-ready visualization.
