# FAQ

## Is this only for Codex?

No. It includes adapter files for several agent environments. `SKILL.md` remains the source of truth.

## Is the output editable?

Yes. PPTX uses native shapes, SVG uses vector/text elements, Draw.io uses editable diagram cells, and Excalidraw uses scene JSON.

## Why ask users to choose format, layout, and style?

Because "editable" can mean PPTX, SVG, Draw.io, Excalidraw, Mermaid, HTML, Markdown, or JSON. Guessing is error-prone.

## Can it make paper figures?

Yes, especially method frameworks, algorithm pipelines, evidence-linked figures, and thesis/proposal technical routes. Users should still verify journal-specific formatting rules.

## Can it make advertising routes?

Yes. It supports campaign strategy, creative production, audience-message-channel maps, and KPI feedback loops.

## Does it execute analyzed project code?

No by default. Source files are treated as untrusted evidence.

# 常见问题

## 只能在 Codex 中用吗？

不是。仓库中已经包含多个 agent 适配文件。

## 输出真的可编辑吗？

是。PPTX、SVG、Draw.io、Excalidraw 都保留可编辑结构，不是截图。

## 为什么必须询问用户选项？

因为“可编辑”“论文”“展示”等词不能唯一决定输出格式、版式和风格。明确询问可以减少错误。
